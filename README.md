# VIFD-LLM Tourism Decision Dynamics

## Overview
This repository provides a research-grade implementation of a Variational Information Flow Dynamics (VIFD) framework for modeling tourism decision-making as a dynamic probabilistic process. The approach integrates drift–diffusion principles with Large Language Model (LLM)-guided policy shaping to simulate and analyze decision trajectories under uncertainty and constraints.

## Key Features
- Variational decision dynamics modeling (drift + diffusion + optimization)
- LLM-guided policy operator (projection + entropy shaping)
- Trajectory-based evaluation (entropy, feasibility, regret, stability)
- Pareto trade-off analysis
- Phase regime and invariant analysis
- Causal decomposition of decision mechanisms
